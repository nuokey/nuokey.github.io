Версии

Актуальная версия

  • Build 1902

    Список изменений:

    1. Добавлена анимация прицеливания для пистолета
    2. Добавлено ограничение по патронам
    3. Добавлена перезарядка (без анимации)
    4. Пистолет появляется в начале игры не в руках персонажа, а рядом, на земле
    5. На уровень добавлена стена
    6. Добавлена система жизней для игрока и противника
    7. Для убийства противника теперь необходимо несколько выстрелов
    8. После нескольких ударов противника игрок умирает, и появляется экран смерти
    9. Количество противников сокращено до 1
    10. Скайбокс заменен на статичную картинку

Устаревшие версии

  • Build 2301

    Список изменений:

    1. Добавлена анимация выстрела
    2. При прицеливании игрок замедляется
    3. Увеличено количество противников до 5

  • Build 2201

    Список изменений:

    1. Изменена модель пистолета
    2. Добавлена задержка после выстрела
    3. Переработано движение игрока
    4. Добавлено движение противника к игроку
    5. Количество противников уменьшено до 1
    6. При игре скрывается мышь
    7. Уменьшена чувствительность
